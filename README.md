# nemo

## Structure

- nemo:
  - `nemo/`: python package

- uv project:
  - `.venv`: virtual environment
  - `.python-version`: python version
  - `pyproject.toml`: project dependencies and description
  - `uv.lock`: lock file

- cdk project:
  - `cdk.json`: cdk project configuration
  - `bin/`: cdk entrypoint
  - `lib/`: cdk stacks

- docs:
  - See [docs](docs/README.md)
